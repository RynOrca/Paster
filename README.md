# Paster：安卓端自动复制截图的工具

由于本人需要频繁用平板的Obsidian做笔记，需要经常在课件截图然后弄进Obsidian里，苦于每次操作一张截图都要跳到相册，再粘贴进Obsidian，于是用AI写了这个小工具。

适用于：想要用安卓平板做笔记，频繁需要截图并且将截图复制粘贴进笔记软件的场景

## 使用方法

需要申请存储权限、通知权限与悬浮窗权限。

打开应用后，点击显示悬浮窗，用系统自带的截图，然后点击悬浮窗，就能把最近一次截图复制到剪贴板，省去了跑去相册，找复制进剪贴板的繁琐过程。
